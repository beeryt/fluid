# Fluid
Fluid is a demonstration of Lattice Boltzmann fluid simulation.

## Installation
You can use cmake to build fluid.
```bash
mkdir build && cd build
cmake ..
make -j8
```

## Usage
```bash
fluid [N] [M]
```
